==============================
        To-Do List App
==============================

Description:
A console-based To-Do List application where
users can add, view, mark, and remove tasks.
Tasks are stored in a JSON file for persistence.

Features:
- Add new tasks
- View all tasks
- Mark tasks as done
- Remove tasks
- Saves tasks in tasks.json

How to Run:
1. Save the file as 01_main.py
2. Open VS Code terminal.
3. Run the program with:
   python3 01_main.py
