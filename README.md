# To-Do List using Python

A command-line To-Do List built using Python. This project helps users organize and manage their daily tasks through a simple menu-driven interface.

The application allows users to add tasks, view existing tasks, mark tasks as completed, and remove tasks. Task data is automatically stored in a JSON file, allowing information to persist between sessions.

## 📌 Project Overview

This project is designed to simulate a lightweight task management application that runs entirely in the terminal.

The program stores tasks locally, making it easy to manage personal tasks without requiring a database or external software.

This project is suitable for beginners who want to practice Python fundamentals and file handling concepts.

## 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Remove tasks
* Automatic task ID generation
* Persistent data storage using JSON
* Interactive menu-driven interface
* Input validation
* Beginner-friendly implementation
